# password-generator
a simple and secure password generator. It generates random passwords, making it suitable for protecting accounts, applications, and personal data

#Features

generating passwords with 8 characters.  
  - uppercase letters
  - lowercase letters  
  - digits  
  - special symbols
  - Lightweight and easy to run from the terminal
